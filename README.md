# 📊 Sales Performance Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Sales Performance Dashboard developed in Microsoft Power BI.

The dashboard transforms raw sales data into meaningful business insights by analyzing revenue, customer behavior, product performance, geographical sales, payment methods, membership tiers, and monthly sales trends.

It enables users to monitor key business KPIs and make data-driven decisions through dynamic visualizations and filters.
